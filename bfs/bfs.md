1: 始点から距離dの頂点集合Vdを求める。

Vdの頂点と隣接

始点からの距離がd以下ではない（訪問済みには印をつける）

入力はグラフG = (V, E)と始点s∈v.

点を基準に扱う

```
for(各頂点　v ∈ V)
	頂点にまだ訪問していないという印をつける。
	//大体bool値でfalse
始点に訪問したという印をつける。
キューQをQ = [s]として初期化
while(キューQが空でない){
	キューQから頂点uをとりだす
	for(uの各隣接頂点v ∈ adj(u){
		if(vがまだ訪問されていない){
			vに訪問したという印をつける
			vをキューQに入れる
		}
	}
}
```
