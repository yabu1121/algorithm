辺重みが等しいなら単一始点最短経路問題はbfsでO(m + n)を実現できる

単一始点最短経路問題が汎用性高い


```
for(各頂点　v ∈　V)D[v] = ∞　
X = φ, D[s] = 0
while(X !=　V){
	V - X の中でDの値が最小の頂点をuとする	
	x = x U{u}とする
	for (頂点uに接続する各辺(u, v)s.t. v ∈ v-x)
		D[v] = min(D[u] + c(u, v))
}
```
